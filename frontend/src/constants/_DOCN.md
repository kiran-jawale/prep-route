# Constants Documentation

## Purpose

The constants folder stores application-wide static values.

---

## Files

| File       | Purpose                                  |
| ---------- | ---------------------------------------- |
| config.ts  | Runtime configuration                    |
| storage.ts | Local storage keys and storage constants |

---

## Responsibilities

* Centralized configuration
* Avoid hardcoded values
* Shared application constants

---

## Notes

No business logic exist inside constants.
